# zamus

## Installation

```sh
go get -u github.com/onedaycat/zamus/...
```

## Deploy

Download lambda in `Release`

```shell
curl -L https://github.com/onedaycat/zamus/releases/download/v0.80.0/dlq.zip -o dlq.zip  
```

Use Terraform

```
terraform apply
```

## CLI

```sh
go get -u github.com/onedaycat/zamus/zamus
```
